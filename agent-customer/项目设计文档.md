## 3.3 Executor（执行器）

```java
public class Executor {

    public Object execute(Plan plan, ExecutionContext context) {

        for (Step step : plan.getSteps()) {
            Object result = toolRegistry.execute(step, context);
            context.put(step.getName(), result);
        }

        return context.getFinalResult();
    }
}
```
