# 3️⃣ Agent 核心（最重要）

```java
@Service
public class AgentService {

    @Autowired
    private Planner planner;

    @Autowired
    private Executor executor;

    public String handle(String input, Long userId) {

        // 1. 生成计划
        Plan plan = planner.plan(input);

        // 2. 执行上下文
        ExecutionContext context = new ExecutionContext(userId);

        // 3. 执行任务
        Object result = executor.execute(plan, context);

        return result == null ? "处理完成" : result.toString();
    }
}
```

# 4️⃣ Planner（可先Mock，后接LLM）

```java
@Service
public class Planner {

    public Plan plan(String input) {

        // ⚠️ MVP阶段先写死，后续替换为 LLM

        List<Step> steps = new ArrayList<>();

        if (input.contains("订单")) {
            steps.add(new Step("queryOrders"));
        }

        if (input.contains("投诉")) {
            steps.add(new Step("complainUser"));
        }

        return new Plan(steps);
    }
}
```

# 5️⃣ Executor（执行引擎）

```java
@Service
public class Executor {

    @Autowired
    private ToolRegistry toolRegistry;

    public Object execute(Plan plan, ExecutionContext context) {

        Object lastResult = null;

        for (Step step : plan.getSteps()) {

            Tool tool = toolRegistry.getTool(step.getAction());

            if (tool == null) {
                throw new RuntimeException("Tool not found: " + step.getAction());
            }

            lastResult = tool.execute(step, context);

            context.put(step.getAction(), lastResult);
        }

        return lastResult;
    }
}
```

# 7️⃣ ToolRegistry

```java
@Component
public class ToolRegistry {

    private final Map<String, Tool> toolMap = new HashMap<>();

    @Autowired
    public ToolRegistry(List<Tool> tools) {
        for (Tool tool : tools) {
            toolMap.put(tool.getClass().getSimpleName(), tool);
        }
    }

    public Tool getTool(String name) {
        return toolMap.get(name + "Tool");
    }
}
```

---

### Step

```java
public class Step {

    private String action;

    public Step(String action) {
        this.action = action;
    }

    public String getAction() {
        return action;
    }
}
```

---

### ExecutionContext

```java
public class ExecutionContext {

    private Long userId;

    private Map<String, Object> data = new HashMap<>();

    public ExecutionContext(Long userId) {
        this.userId = userId;
    }

    public void put(String key, Object value) {
        data.put(key, value);
    }

    public Object get(String key) {
        return data.get(key);
    }

    public Long getUserId() {
        return userId;
    }
}
```
